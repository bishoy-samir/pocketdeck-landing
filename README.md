# PocketDeck

**A fast, controller-first game launcher for Windows handhelds.**

🔗 Website: https://bishoy-samir.github.io/pocketdeck-landing/
🔗 Mirror: https://pocketdeck.prog-bishoy15.workers.dev/

## What is PocketDeck?

PocketDeck is a lightweight game launcher built specifically for Windows-based handheld PCs — devices like the ASUS ROG Ally / ROG Xbox Ally X, Lenovo Legion Go, MSI Claw, and Ayaneo. Instead of digging through flat lists or a desktop built for a mouse and keyboard, PocketDeck gives you a real 3D coverflow library browser, full controller navigation, automatic cover art and metadata, and a UI designed from the ground up to be used comfortably on a small screen with a gamepad in hand.

It's not a desktop launcher with controller support bolted on afterward — every screen (browsing, game details, library management, profile, settings) is designed to be navigated entirely with a controller, with mouse/keyboard as a fallback rather than the primary input.

## Why I built this

I'm a solo/small developer, and I made PocketDeck because I wanted something that actually felt built for a handheld — not a shrunken desktop app. Most existing launchers are designed mouse-first and just barely usable with a controller. PocketDeck flips that: controller navigation, big readable UI, and smooth coverflow browsing come first, and everything else is built around that.

This is an independent project, actively developed and improved based on real feedback from handheld owners. If you run into bugs, have a feature request, or just want to say what's working/not working for your device, I want to hear about it.

## Features

- Real 3D coverflow library browsing — smooth, physics-based, not a flat carousel
- Full controller navigation across every screen (browse, sort, profile, details, launch)
- Automatic cover art, artwork, genres, ratings, and synopsis via the RAWG and IGDB APIs
- Manual add/edit for games when you'd rather enter details yourself
- Multiple curated themes, including gradient options
- Optimized for low memory footprint and limited handheld hardware

## Requirements

- Windows 10/11 (64-bit)
- Works on Windows handhelds (ROG Ally, ROG Xbox Ally X, Legion Go, MSI Claw, Ayaneo, etc.) as well as regular Windows desktops/laptops
- Not compatible with Steam Deck's default SteamOS — only if it's running Windows
